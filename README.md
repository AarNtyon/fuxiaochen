## 启动项目

```shell
# 1. 安装依赖
$ pnpm i

# 2. 准备数据库
$ pnpm db:push

# 3. 生成admin用户，默认相关配置在 .env.development 中
$ pnpm db:seed

# 4. 启动开发环境
$ pnpm dev
```
